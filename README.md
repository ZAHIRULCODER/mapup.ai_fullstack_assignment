### Frontend

1. **Update Render URLs**:
   - Replace all Render URLs in the codebase with `http://localhost:3000` to ensure they point to local server (backend code).

2. **Install Dependencies**:
   ```bash
   npm install

2. **Start the Development Server**:
   ```bash
   npm run dev
